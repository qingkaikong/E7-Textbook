Chapter 11: Visualization and Plotting¶

Visualizing data is usually the best way to convey important engineering ideas and information, especially if the information is made up of many, many numbers. The ability to visualize and plot data quickly and in many different ways is one of Python’s most powerful features.

Python has numerous graphics generators that enable you to efficiently display plots, surfaces, volumes, vector fields, histograms, animations, and many other data plots. By the end of this chapter, you should be familiar with the most common ones and have enough information to explore the rest.

11.1 2D Plotting¶

The basic plotting function in Python comes from the matplotlib.pyplot library. We can call a plot(x, y) function that can provide a clean visual display of two arrays of data, x and y.

EXAMPLE 11.1: Basic Plot

import numpy as np
import matplotlib.pyplot as plt

# Example plot of x against y
xdata = np.array([0, 1, 2, 3])
ydata = np.power(xdata, 2)
plt.plot(xdata, ydata) # Plots x and y

[<matplotlib.lines.Line2D at 0x66f46f0>]

You will notice that this function connects each point with a blue line. To make the function look more “smooth”, we should use a finer discretization of points. See the following example for a smoother example of the previous curve:

EXAMPLE 11.2: Smooth Plot

import numpy as np
import matplotlib.pyplot as plt

xdata = np.linspace(0, 3, 100) # 100 points instead of 3!
ydata = np.power(xdata, 2)
plt.plot(xdata, ydata) # Smooth!

[<matplotlib.lines.Line2D at 0x6731930>]

EXAMPLE 11.3: Plot Customization

import numpy as np
import matplotlib.pyplot as plt

xdata = np.linspace(-5, 5, 100)
ydata = np.power(xdata, 3) # Plot f(x) = x^3
plt.plot(xdata, ydata, 'g--') # Green dotted line!

[<matplotlib.lines.Line2D at 0x69bdff0>]

It is customary in engineering to always give your plot a title and axis labels. The plt.title function is used for the plot title, and the plt.xlabel and plt.ylabel functions are used for axes labels.

EXAMPLE 11.4: Plot Customization with Titles

import numpy as np
import matplotlib.pyplot as plt

xdata = np.linspace(-5, 5, 100)
squared = np.power(xdata, 2) # Plot f(x) = x^2
cubed = np.power(xdata, 3) # Plot f(x) = x^3
plt.plot(xdata, squared, 'r+-') # + marks as points, red solid line
plt.plot(xdata, cubed, 'ko-') # circles as points, black solid line
plt.title('Plot of Various Polynomials')
plt.xlabel('x-axis')
plt.ylabel('y-axis')

Text(0,0.5,'y-axis')

It is often necessary to add a legend to your plot as well. We use the plt.legend function to do this. We add the label as an extra argument to plt.plot as shown below, and specify the location, such as upper right as an input to plt.legend.

_EXAMPLE 11.5: Adding a Legend

import numpy as np
import matplotlib.pyplot as plt
from math import pi

xdata = np.linspace(-2*pi, 2*pi, 200)
sin_x = np.sin(xdata)
cos_x = np.cos(xdata)

plt.plot(xdata, sin_x, '-', label='sin(x)') # Label is shown in legend
plt.plot(xdata, cos_x, '-', label='cos(x)')
plt.title('Plot of Trigonometric Functions')
plt.xlabel('x-axis')
plt.ylabel('y-axis')
plt.legend(loc='upper right')

<matplotlib.legend.Legend at 0x44d7f10>

Python automatically sets an appropriate range for the axes that is justifiable in most cases, but suppose we want to manually specify the axes limits. We do this as follows:

axes = plt.gca() # gca = get current axes
axes.set_xlim([xmin, xmax]) # add your limits in place of xmin and xmax
axes.set_ylim([ymin, ymax])

_EXAMPLE 11.6: Specifying Axes Limits

Sometimes it is desirable to have two graphs in one figure. We do this using subplots.

The function subplot(num_rows, num_cols, index) should be called before the plot function is called. Consider the example below.

_EXAMPLE 11.7: Subplots

import numpy as np
import matplotlib.pyplot as plt
from math import pi

x1 = np.linspace(0, 6, 100)
y1 = np.exp(-x1) * np.sin(2 * pi * x1)

x2 = np.linspace(0, 2, 100)
y2 = np.sin(pi * x2)

plt.subplot(2, 1, 1) # 2 by 1 subplot - the 1st (top) plot
plt.plot(x1, y1, '-')
plt.title('Damped vs. Undamped Oscillation')
plt.xlabel('Time')
plt.ylabel('Position')

plt.subplot(2, 1, 2) # 2 by 1 subplot - the 2nd (bottom) plot
plt.plot(x2, y2, '-')
plt.xlabel('Time')
plt.ylabel('Position')

Text(0,0.5,'Position')

The final common set of plots that are often generated in science and engineering are the semilog and log-log plots. We simply call the semilogx or semilogy functions to specify the axis that should have a logarithmic scale, or loglog for a log-log plot.

EXAMPLE 11.8: Semilong and Log-Log Plots

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 5, 6)
y = np.array([1, 2, 4, 8, 16, 2])

plt.subplot(1, 2, 1)
plt.semilogy(x, y, '-')
plt.ylabel('Semilog y')
plt.title('Semilog')

plt.subplot(1, 2, 2)
plt.loglog(x, y, '-')
plt.title('Log-Log')

Text(0.5,1,'Log-Log')
