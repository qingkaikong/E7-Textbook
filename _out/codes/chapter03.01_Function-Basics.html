TRY IT! Compute the function myTrigSum for a=2 and b=3. Assign the first output argument to the variable C and the second output argument to the variable D.

import math
def myTrigSum(a, b):
    A = math.sin(a) + math.cos(b)
    B = math.sin(b) + math.cos(a)
    return [A, B]

[C, D] = myTrigSum(2, 3)
print("C =", C)
print("D =", D)

C = -0.0806950697747637
D = -0.2750268284872752
