Chapter 5: Iteration¶

Many tasks in life are boring or tedious because they require doing the same basic actions over and over again—iterating—in slightly different contexts. For example, consider looking up the definition of 20 words in the dictionary, populating a large table of numbers with data, alphabetizing many stacks of paper, or dusting off every object and shelf in your room. Since repetitive tasks appear so frequently, it is only natural that programming languages like Python would have direct methods of performing iteration.

This chapter teaches you how to program iterative tasks. With branching and iteration, it is possible to program just about any task that you can imagine.

5.1 For-Loops¶

A for-loop is a set of instructions that is repeated, or iterated, for every value of the sequence specified. Sometimes for-loops are referred to as definite loops because they have a predefined begin and end.

Initially, the for-loop assigns the looping variable, or the variable that holds the current value of the sequence to be iterated, to the first element of the sequence. Then, it assigns the looping variable to the next element of the sequence and executes the code block again. It continues until there are no more elements in the sequence.

The general syntax for a for-loop block is as follows.

CONSTRUCTION: For-loop¶

for looping variable in <sequence>:
    # <CODE BLOCK TO BE REPEATED>

Sequences can be many things. Perhaps the most common is a set of integers between a start and end value, written as range(start, end). Consider the following example:

EXAMPLE 5.1: Basic For-Loop

n = 0
for i in range(1, 4):
    n = n + i

IMPORTANT! The range(start, end) is inclusive of start and exclusive of end. So, for example, for i in range(0, 4) will start iteration at i = 0 and end at i = 3!

WHAT IS HAPPENING? First, recall that range(1, 4) is the set of numbers beginning at 1 and ending at 3.
1) The variable n is assigned the value 0.
2) The variable i is assigned the value 1.
3) The varaiable n is assigned the value n + i (\(0 + 1 = 1\)).
4) The variable i is assigned the value 2.
5) The varaiable n is assigned the value n + i (\(1 + 2 = 3\)).
6) The variable i is assigned the value 2.
7) The varaiable n is assigned the value n + i (\(1 + 2 = 3\)).
8) With no more values to assign in the looping array, the for-loop is terminated with n = 6.

def sum_elements(a):
    """Returns the sum of all the elements of the list a."""
    sum = 0
    for elem in a:

Note: A Python shortcut that is commonly used is the += shortcut. In Python and many other programming languages, a statement like i += 1 is equivalent to i = i + 1 and is often preferred.

EXAMPLE 5.4: Given a list of integers, lst, add only the even elements of the list. Recall a number is even if it can be divided by 2 without remainder.

def sum_only_even(lst):
    """Returns the sum of the even elements of the list lst."""
    sum = 0
    for elem in lst:
        if elem % 2 == 0: # Even check!
            sum += elem
        # If elem is not even, nothing happens to sum and the next iteration of the loop begins.
    return sum

sum_only_even([2, 3, 4, 6]) # 2 + 4 + 6 = 12

Just like if-statements, for-loops can be nested.

def print_chars_in_word(word_list):
    """Prints characters of the words in word_list one by one using nested
    loops."""
    for word in word_list: # loops over each word in the list
        for c in word: # Loops over each character in a word
            print(c)

print_chars_in_word(['apple', 'egg'])

WARNING! Although possible, try not to change the looping variable inside of the for-loop. It will make your code very complicaed and will likely result in errors. An example of an improper implementation is below:

EXAMPLE 5.6: Sum Squares

def sum_squares(n):
    """Sums the squares of the integers from 1 to n."""
    total = 0
    for i in range(1, n + 1):
        i = i**2 # The looping variable i is changed!
        total += i
    return total

Here is a proper implementation of the same function that does not change the looping variable.

def sum_squares(n):
    """Sums the squares of the integers from 1 to n."""
    total = 0
    for i in range(1, n + 1):
        total += i**2 # Does not change the value of the looping variable i
    return total
