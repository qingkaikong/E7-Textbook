EXAMPLE: Illustration of Case 1. We pick a matrix A and a vector y such that y is not in the range of A. Therefore, there is no solution to the matrix equation.

import numpy as np
from numpy.linalg import matrix_rank

A = np.array([[1, 2, 3], [0, 3, 1], [-1, 14, 7]])
A = np.matrix(A)
print(A)

y = np.array([[1], [2], [3]])
y = np.matrix(y)
print(y)

print(matrix_rank(A))
print(matrix_rank(np.concatenate((A, y), axis=1)))

EXAMPLE: Illustration of Case 2. We pick a matrix A and a vector y such that y is in the range of A, and such that the rank of A is equal to the number of unknowns.

from numpy.linalg import inv

A = np.array([[1, 2, 3], [0, 3, 1], [-1, 14, 7]])
A = np.matrix(A)
print("A =", A)
print()

y = np.array([[1], [2], [3]])
y = np.matrix(y)
print("y =", y)
print()

print(matrix_rank(A))
print(matrix_rank(np.concatenate((A, y), axis=1)))
print()

x = inv(A)*y
print("x =", x)
print()

EXAMPLE: Illustration of Case 3. We first illustrate this case with a square matrix.

from scipy.linalg import null_space

A = np.array([[1, 2, 3, 0], [0, 3, 1, 0], [-1, 14, 7, 0], [0, 0, 0, 0]])
A = np.matrix(A)
print("A =", A)
print()

y = A * np.matrix(np.array([[1], [1], [1], [0]]))
print("y =", y)
print()

x = np.linalg.pinv(A)*y
print("x =", x)
print()

print("A * x =", A * x)
print("A * [0; 0; 0; 1] =", A * np.matrix(np.array([[0], [0], [0], [1]])))
print("A * (x + [0; 0; 0; 100]) =", A * (x +  np.matrix(np.array([[0], [0], [0], [100]]))))

# To investigate the example of a nonsquare matrix, let us consider the following script:
# Define A and y

print("y =", y)
