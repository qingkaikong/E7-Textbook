EXAMPLE: Consider the following function:

def myWorkspaceTest(a, b):
    x = a + b
    y = x * b
    z = a + b
    return [x, y]

TRY IT! What will the value of a, b, x, y, and z be after the following code is run?

a = 2
b = 3
z = 1
[y, x] = myWorkspaceTest(b, a)    # y = 5, x = 10
print("z =", z)
print("y =", y)
print("x =", x)

z = 1
y = 5
x = 10
