TRY IT! Again, the \(sqrt{2}\) is the root of the function f (x) = \(x^2−2\). Using x0 = 1.4 as a starting point, use the previous equation to estimate \(sqrt{2}\). Compare this approximation with the value computed by MATLAB’s sqrt function.

import math

f = lambda x: math.pow(x, 2) - 2
f_prime = lambda x: 2*x
newton_raphson = 1.4 - (f(1.4))/(f_prime(1.4))

print("newton_raphson =", newton_raphson)
print("sqrt(2) =", math.sqrt(2))

newton_raphson = 1.4142857142857144
sqrt(2) = 1.4142135623730951

TRY IT! Write a function with header myNewton(f, df, x0, tol), where R is an estimation of the root of f, f is a handle to the function f (x), df is a handle to the function f'(x), x0 is an initial guess, and tol is the error tolerance. The error measurement should be | f (x) |.

def myNewton(f, df, x0, tol):
    # output is an estimation of the root of f using the Newton Raphson method
    # recursive implementation
    if abs(f(x0)) < tol:
        return x0
    else:
        return myNewton(f, df, x0 - f(x0)/df(x0), tol)
